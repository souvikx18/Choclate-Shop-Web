## 🌟 Overview
Welcome to **A Little Chocolate Shop (LCS)**! This project represents a fully functional, multi-page frontend website designed to showcase a local business. Built with a mobile-first approach, it features smooth navigation, a clean and professional UI, interactive elements, and modern typography to deliver a premium browsing experience. 

## ✨ Features
- **📱 Fully Responsive Layout**: Adapts beautifully to mobile, tablet, and desktop screens.
- **🗺️ Multi-Page Navigation**: Includes Home, About, Menu, Store Hours, and Contact Us pages smoothly connected together.
- **🍔 Interactive Mobile Menu**: A custom-built hamburger menu using Vanilla JavaScript for mobile users.
- **🖱️ Smooth Scrolling**: Enhanced user experience with smooth anchor link navigation for the menu and about sections.
- **♿ Accessible & Semantic HTML**: Uses modern HTML5 tags for better SEO, meaningful structure, and accessibility.
- **🎨 Modern Styling**: Styled entirely from scratch with custom CSS, utilizing Flexbox, Gradients, and Google Fonts (`Poppins`).

## 📂 Project Structure
```text
📦 Choclate-Shop-Web
 ┣ 📜 index.html         # Main landing page (Home, About, History, Menu)
 ┣ 📜 contact.html       # Contact form and interactive location details
 ┣ 📜 hours.html         # Store operation hours & schedule
 ┣ 📜 style.css          # Main stylesheet for the application
 ┣ 📜 hours_style.css    # Specific styles for the hours page
 ┗ 🖼️ Assets (Images & Logos: logo.png, shop_img.jpeg, choclate_img.jpeg)
```

## 🛠️ Built With
- **HTML5**: Leveraged semantic tags (`<header>`, `<main>`, `<figure>`, `<article>`), accessible tables, and forms.
- **CSS3**: Implemented modern custom variable colors, responsive media queries, CSS Flexbox layouts, beautiful gradients, and component scoping.
- **JavaScript (Vanilla)**: DOM manipulation for the responsive navbar logic and smooth anchor scroll behaviors.

## 🚀 Getting Started
To view this project locally on your machine, follow these simple steps:
1. Clone this repository or download the ZIP file.
2. Unzip/extract the contents to your desired folder.
3. Open the folder in your file explorer.
4. Double-click on `index.html` to open it in your default web browser. **No local server required!**

## 💡 What I Learned
As my very first front-end project, this was a fantastic hands-on journey into web development. Throughout this build, I learned how to:
- Structure a semantic, scalable multi-page website correctly.
- Implement robust responsive design without relying on any external CSS frameworks (like Bootstrap or Tailwind).
- Use Vanilla JavaScript to add interactivity to static UI components.
- Design fully customized forms, tables, and typography hierarchy.
